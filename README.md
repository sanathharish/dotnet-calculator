# .NET Calculator

A simple console-based calculator built in C# using clean OOP principles.

## Features
- Add, Subtract, Multiply, Divide
- Input validation
- Loop-based menu
- Encapsulation and interfaces
- Enum-driven operations

## Design Principles
- Separation of concerns
- Encapsulation
- Interface-based design
- Future-ready for Web API conversion

## Tech Stack
- .NET (Console App)
- C#
- Git & GitHub

## Future Improvements
- Unit tests
- ASP.NET Core Web API
- Frontend client
